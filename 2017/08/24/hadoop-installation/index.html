Hadoop笔记(二)：Hadoop的安装
安装Hadoop之前搜了很多关于Hadoop是什么的资料，大部分的资料都将Hadoop讲得神乎其神，然而看了这些文字之后仍然不懂Hadoop到底能干什么样的事情。于是我想要从例子出发，理解Hadoop到底是怎么样工作的。本节内容讲的是如何在Mac上安装Hadoop并且配置Eclipse，以及运行MapReduce的“hello world”程序–WordCount.
1.安装Hadoop：
安装的教程在网上都能找到,我参考的文章是：http://blog.csdn.net/u012336567/article/details/50778989。
安装Hadoop之前需要安装ssh，因为安装Hadoop需要远程登录。Mac OS X在“系统偏好设置”“共享”中的“远程登录”勾选ssh就可以了。然后运行下面命令行确认是否需要密码登录localhost：
$ ssh localhost
设置ssh免密码登录：
$ ssh-keygen -t dsa -P ‘’ -f ~/.ssh/id_dsa
第一行：ssh-keygen代表生成密钥，-t指定生成的密钥类型，dsa代表dsa密钥认证；-p提供密语，-f指生成的密钥文件
第二行：将公钥加入到用于认证的公钥文件中
$ cat ~/.ssh/id_dsa.pub >> ~/.ssh/authorized_keys
接下来就可以安装Hadoop了。主要的命令行是:
$ brew install hadoop(安装Hadoop)在Mac上不需要自己配置Java_Home因为brew在安装时已经配置好。
安装之后还要修改配置文件。配置过程参见上面的链接。
2.运行Hadoop：
跳转目录/usr/local/Cellar/hadoop/2.8.0/bin下：
$ hadoop namenode -format
再进入/usr/local/Cellar/hadoop/2.8.0/sbin,执行：
$ ./start-all.sh
访问一下网址可以查看Hadoop是否成功运行：
Resource Manager: http://localhost:50070
JobTracker: http://localhost:8088
Specific Node Information: http://localhost:8042
3.配置Eclipse:
下载puglins，在finder中右击eclipse图标，选择“显示包内容”，进入文件夹，将下载的压缩包解压，之后将其中的2.6.0版本的jar包拷贝到plugins文件夹中。此时，打开eclpse的“偏好设置”，发现里面有了Hadoop Map/Reduce选项，点开后，将/usr/local/Cellar/hadoop/2.8.0/libexec拷贝到Hadoop installation directory中。建立Hadoop项目就可以了。
4.运行WordCount程序:
WordCount代码网上可以找到，我现在只试着运行代码看结果，之后再去理解代码。运行的步骤如下：
在新建的Hadoop项目中建立class，将代码复制到class中，然后导出jar包(也可以直接运行程序，但是要输入参数，本文只讲导出jar包在命令行执行的过程)。然后将jar包放到主目录下，即/Users/**/,在后面的命令行中都要带着路径。执行一下命令行：
$ hadoop fs -mkdir /user/input
$ hadoop fs -put /usr/local/Cellar/hadoop/2.8.0/README.txt /input(将README.txt拷贝到input文件夹中，一定要添加路径)
$ hadoop jar /Users/apple/WordCount.jar hadoop2.WordCount /user/input output（运行程序，一定要添加各种路径,hadoop2是我的package名称,WordCount是class的名称）
$ hadoop fs -text output/part-r-00000(程序运行结束后，查看运行结果，结果是每个词的数量)
