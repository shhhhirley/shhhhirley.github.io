两道动规题，都可以用复杂的非动规实现，做第一道的时候不会，应该是看了答案，做第二道的时候不知道要用动规，做的很艰难，很多小细节都要考虑到。

Longest palindromic substring:

create a boolean 2-dimensional array to keep the status of a substring with different beginnings and ends. For example, dp[i][j] is to represent if the substring starting at i, ending with j is a palindromic substring. Then, check if d[i+1][j+1] is palindromic by judging if d[i][j] && s[i+1] == s[j+1].

Trapping Rain Water

首先是我自己写的，不用动规的。先记录差值，寻找极大值点。再从极大值点向右找到第一个比该点大的点，容量就是极大值点减去中间的值。如果没找到比极大值点大的点，从最右往左找最大的点作为容器的高度。实现过程有很多复杂的小细节，太麻烦了！

大神解法：

  1. 两次遍历，一次存左边最大，一次存右边最大，然后比较左右中较小的作为容器的高度，计算存水量。
  1. 一次遍历即可，从两边往中间遍历。如果左值小于右值，右值不变，左值右移，如果最左点往右找到比最左值小的值，存水；否则，将比最左值小的值作为最左值。再次比较左右值大小。如果左值大于右值，右值左移，如果找到比最右值小的值，存水；否则，将比最右值大的值作为最右值。再次比较左右值大小。直到左右遇到。
